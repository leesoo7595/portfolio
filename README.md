# portfolio

## Description

- 자바스크립트만을 활용하여 portfolio라는 프로젝트를 만들어보았다.
- 간단한 CRUD형 게시판 블로그
- 백엔드 NodeJs 안에 프론트 React로 구성되어있다.

## Features

- 게시판에 글쓰기 및 등록 가능
- 등록 후 데이터베이스(mysql) 저장 가능
- 게시판 리스트 보여주기 가능
- 글쓰기 + 이미지 등록 가능
- 이미지 S3에 업로드 가능

## Tech

- Javascript
- NodeJs
- React
- AWS Elastic Beanstalk
